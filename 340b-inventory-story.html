What This Story Delivers

VIEW

Split Inventory View

Every product shows standard vs 340B stock side-by-side — quantities, lot counts, earliest expiry per source.

LOGIC

Replenishment Assessment

Computed per-product: is 340B stock below reorder point? Zero stock? Expiring soon? How many dispensed since last receipt?

DATA

Acquisition Source Tracking

New acquisition_source column on InventoryLot — tags every lot as "standard", "340b", or "wac" at receiving time.

API

Dedicated Endpoint

GET /inventory/340b — returns summary stats + per-product breakdown with filters for "only 340B" and "needs replenishment".

Layer 3: Service

New method on StockService — the heaviest piece of work. Two SQL queries + in-memory aggregation.

File: stock_service.py

Step 3a: Method signature▶
async def get_340b_inventory( self, actor_id: str, only_with_340b: bool = False, only_needing_replenishment: bool = False, ) -> dict[str, Any]: """Get full 340B vs standard inventory breakdown with replenishment needs."""
Step 3b: Query 1 — Stock grouped by product + source▶
# Aggregate stock per product per acquisition_source stmt = ( select( InventoryProduct.id, InventoryProduct.ndc, InventoryProduct.name, InventoryProduct.manufacturer, InventoryProduct.is_controlled, InventoryProduct.reorder_point, InventoryProduct.reorder_quantity, InventoryLot.acquisition_source, func.sum(InventoryStock.quantity_on_hand).label("qty_on_hand"), func.sum(InventoryStock.quantity_reserved).label("qty_reserved"), func.sum(InventoryStock.quantity_available).label("qty_available"), func.count(distinct(InventoryLot.id)).label("lot_count"), func.min(InventoryLot.expiration_date).label("earliest_expiry"), ) .join(InventoryStock, InventoryStock.product_id == InventoryProduct.id) .outerjoin(InventoryLot, InventoryStock.lot_id == InventoryLot.id) .where(InventoryProduct.is_active.is_(True)) .group_by(InventoryProduct.id, InventoryLot.acquisition_source) )

Returns one row per (product, acquisition_source) pair. Pivot in Python to build standard_stock + stock_340b per product.

Step 3c: Query 2 — Replenishment data▶
# Last 340B receipt date + dispensed since, per product receipt_stmt = ( select( InventoryTransaction.product_id, func.max(InventoryTransaction.created_at).label("last_receipt"), ) .join(InventoryLot, InventoryTransaction.lot_id == InventoryLot.id) .where( InventoryTransaction.transaction_type == TransactionType.RECEIPT, InventoryLot.acquisition_source == "340b", ) .group_by(InventoryTransaction.product_id) ) # Then for each product, count dispenses from 340B lots since last receipt
Step 3d: Replenishment logic (in-memory)▶
# For each product, compute replenishment: if qty_340b == 0: reason = "zero_stock" needs_replenishment = True elif qty_340b < product.reorder_point: reason = "below_threshold" needs_replenishment = True elif earliest_340b_expiry and days_until < 30: reason = "expiring_soon" needs_replenishment = True else: reason = None needs_replenishment = False suggested_quantity = product.reorder_quantity
Step 3e: Apply filters + build response▶
# Optional filters before returning if only_with_340b: products = [p for p in products if p.stock_340b.quantity_on_hand > 0] if only_needing_replenishment: products = [p for p in products if p.replenishment.needs_replenishment] # Build summary from filtered list summary = Inventory340BSummary( total_340b_products=len([p for p in products if p.stock_340b.lot_count > 0]), total_340b_units=sum(p.stock_340b.quantity_on_hand for p in products), total_standard_units=sum(p.standard_stock.quantity_on_hand for p in products), products_needing_replenishment=len([p for p in products if p.replenishment.needs_replenishment]), )

export function use340BInventory(params?: { only_with_340b?: boolean; only_needing_replenishment?: boolean; }) { return useQuery<Inventory340BResponse>({ queryKey: ["inventory-340b", params], queryFn: () => { const qs = new URLSearchParams(); if (params?.only_with_340b) qs.set("only_with_340b", "true"); if (params?.only_needing_replenishment) qs.set("only_needing_replenishment", "true"); const q = qs.toString(); return fetchJson<Inventory340BResponse>( q ? `${BASE}/340b?${q}` : `${BASE}/340b` ); }, }); }
